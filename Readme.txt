# Subnetting Web App MCP

Đây là phiên bản mở rộng với hỗ trợ tính toán CIDR và VLSM.

## Hướng dẫn sử dụng
- Nhập địa chỉ mạng dạng CIDR, ví dụ: `192.168.1.0/24`
- Nếu chọn **CIDR**, nhập số subnet mong muốn như `4`.
- Nếu chọn **VLSM**, nhập danh sách số host mỗi subnet tách bằng dấu phẩy như `100,50,20`.
- Nhấn "Tính toán" và kết quả sẽ hiển thị bảng các subnet với network, broadcast, host min/max và mask.
